# 开发和部署

# 准备

首先你需要下载Microsoft Visual Studio 2019或更新的版本，然后你需要下载安装cmake

# 创建新目录

在项目主目录下运行:

md build

# 配置Cmake

进入build目录后执行

cmake ..  -DCMAKE_BUILD_TYPE=Release

# 使用cmake进行编译

在build目录内执行

cmake --build   . --config Release

等待项目编译，等编译完后在Release目录即可生成trapdoor-{mod vesrion}-{game version}.dll文件

注意，只能用Release版本，Debug版会用不了
