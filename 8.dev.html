# 8. 高级 - 开发和部署

# 如何编译

  1. 下载Microsoft Visual Studio 2019或更新的版本，然后你需要下载并安装cmake

  2. 创建并进入build目录,在项目根目录下运行:

    mkdir build && cd build

  3. 构建Cmake项目,在build下运行如下命令:

    cmake ..  -DCMAKE_BUILD_TYPE=Release -DDEV=OFF

  4. 编译项目

    cmake --build   . --config Release

等待项目编译，等编译完后在Release目录即可生成trapdoor-{mod vesrion}-{game version}.dll文件
